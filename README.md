# URL Query String Analyzer

**Free online tool to extract, analyze, and decode query parameters from any URL. Supports shortened URLs, Proofpoint links, redirects, and JSON export.**

[![URL Query Analyzer](https://img.shields.io/badge/URL-Analyzer-blue)](https://url-query-analyzer.vercel.app)

## üöÄ Features

- **URL Parameter Extraction**: Instantly extract and display all query parameters from any URL
- **Shortened URL Support**: Automatically handles bit.ly, tinyurl, and other URL shorteners
- **Redirect Following**: Follows HTTP redirects to get the final destination URL
- **Proofpoint URL Decoding**: Special support for decoding Proofpoint and other campaign URLs
- **Clean Table Display**: Sortable table view with parameter names and values
- **Copy to Clipboard**: One-click copying for parameter values
- **JSON Export**: Export analysis results as JSON for further processing
- **Dark/Light Mode**: Toggle between themes for comfortable viewing
- **Mobile Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Fast & Free**: No registration required, instant results

## üéØ Use Cases

- **Developers**: Debug API calls and analyze URL structures
- **Security Analysts**: Examine suspicious URLs and campaign links
- **Marketers**: Track UTM parameters and campaign performance
- **QA Testers**: Validate URL parameters in web applications
- **Students**: Learn about URL structure and query strings

## üìñ How to Use

1. **Enter URL**: Paste any URL in the input field (supports regular URLs, shortened URLs, and campaign URLs)
2. **Click Analyze**: Hit the "Analyze URL" button or press Enter
3. **View Results**: See all extracted parameters in a clean, sortable table
4. **Copy Values**: Click the copy icon next to any parameter to copy its value
5. **Export Data**: Download results as JSON for offline analysis

### Supported URL Types

- Regular URLs with query parameters
- Shortened URLs (bit.ly, tinyurl, etc.)
- Proofpoint protected links
- Campaign URLs with tracking parameters
- URLs with encoded characters
